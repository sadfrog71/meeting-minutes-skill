# SKILL.md

## Identity
- name: 会议纪要整理
- description: 将原始会议记录整理为结构化、可执行的会议纪要
- triggers: ["会议纪要", "整理会议", "meeting minutes", "会议记录", "会议总结"]

## Instructions

### 输入处理
1. 解析会议基本信息（时间、地点、参会人员、主持人、记录人）
2. 识别会议主题和议题
3. 提取讨论内容和决策事项
4. 识别后续工作安排和责任人

### 输出结构
按以下格式组织输出：

```
## 基本信息
| 项目 | 内容 |
|------|------|
| 时间 | |
| 地点 | |
| 主持 | |
| 记录 | |

## 会议主题
一句话概括会议目的

## 会议结论
按编号列出所有决策事项，使用**加粗**标注关键人和时间节点

## 责任分工
| 人员 | 任务 |
|------|------|
| | |

## 后续工作
按优先级列出行动项，包含截止时间和负责人
```

### 关键规则
- 会议结论必须包含：决策内容、责任人、关键时间节点
- 责任分工要明确到具体人名
- 后续工作要有可执行性（包含行动项+截止时间）
- 使用公司/项目统一的任务分类（如有）

## Constraints
- 只基于提供的原始记录整理，不添加未提及的信息
- 无法确认的信息标注"[待确认]"，不编造
- 涉及人名时保持原始写法
- 时间格式统一为：YYYY年M月D日 H:MM-H:MM
- 附件内容不展开，仅标注"[见附件]"
